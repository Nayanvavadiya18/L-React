# 📦 Inventory Management System

## 📌 Project Description
This project is a full-stack Inventory Management System developed during my React Developer internship.  
It allows users to perform CRUD operations on inventory items and manage data efficiently.

---

## 🚀 Features

- ✅ Add new inventory items  
- ✅ View all items  
- ✅ Update existing items  
- ✅ Delete items  
- ✅ Search items (by name/mobile)  
- ✅ Filter items  
- ✅ Responsive UI  

---

## 🛠️ Tech Stack

### Frontend:
- React.js  
- Axios  
- React Router  
- Material UI (optional)

### Backend:
- Node.js  
- Express.js  
- MongoDB  

---

## 🔗 API Endpoints

| Method | Endpoint | Description |
|--------|--------|-------------|
| POST | `/api/users/create` | Create item |
| GET | `/api/users/` | Get all items |
| GET | `/api/users/:id` | Get single item |
| PUT | `/api/users/:id` | Update item |
| DELETE | `/api/users/:id` | Delete item |
